Glossário de Termos de Acessibilidade

Histórias Inspiradoras Página Inicial Recursos Diretrizes Glossário de Termos

WCAG (Web Content Accessibility Guidelines)

Diretrizes de Acessibilidade para Conteúdo da Web. São padrões desenvolvidos pelo World Wide Web Consortium (W3C) para tornar o conteúdo web mais acessível para pessoas com deficiências.

Contraste de Cor

A diferença de luminosidade entre o texto e o plano de fundo em uma interface. Alto contraste é fundamental para garantir que o texto seja legível para pessoas com baixa visão.

ARIA (Accessible Rich Internet Applications)

Um conjunto de atributos especiais que podem ser adicionados a elementos HTML para melhorar a semântica e acessibilidade de aplicativos da web, especialmente aqueles que utilizam tecnologias AJAX e JavaScript.

Leitor de Tela

Um software que converte texto digital, incluindo informações exibidas na tela, em voz ou braille, permitindo que pessoas com deficiência visual acessem conteúdo digital.

Alt Text (Texto Alternativo)

Um atributo em uma imagem HTML usado para descrever o conteúdo da imagem. É fundamental para usuários de leitores de tela, permitindo que eles entendam o que está na imagem.
